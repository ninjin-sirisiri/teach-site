# デザインのミニマル化・モダナイズ計画

集中しやすい学習環境を提供するため、装飾を抑え、タイポグラフィと余白を重視した「ミニマル・モダン」なデザインに刷新します。

## 1. デザインコンセプトの変更
- **過剰な装飾の排除**: グラデーション、ガラスモーフィズム（透過・ぼかし）、3D回転などの視覚的ノイズを最小限にします。
- **フラット＆ソリッド**: 境界線（Border）と控えめな背景色の使い分けで構造を表現します。
- **一貫性のある色使い**: アクセントカラーを絞り、グレー系を基調とした落ち着いた配色にします。

## 2. 具体的な変更内容

### 2.1 スタイル変数 (style.css)
- `gradient-primary` を控えめなものに変更、またはソリッドカラーに変更。
- シャドウ (`shadow-*`) をよりフラットに、透明度を下げます。
- 余白 (`spacing-*`) を見直し、コンテンツ周りの呼吸を整えます。

### 2.2 サイドバー
- `backdrop-filter` を削除し、ソリッドな背景色（白または非常に薄い灰色）に変更。
- アイコンとテキストの間隔を最適化。
- アクティブ状態の表示を、よりシンプル（例：左側のアクセントラインのみなど）にします。

### 2.3 ヒーローセクション (Hero Section)
- 派手なグラデーション背景を、ライトグレーの背景と細い境界線に変更。
- 画像の3D回転アニメーションを削除し、フラットな配置にします。
- タイトルのフォントサイズを調整し、読みやすさを向上させます。

### 2.4 カードとコンテンツ
- カードのホバー時の「浮き上がり」を最小限（または境界線の色変化のみ）にします。
- コードブロックのデザインを整理し、エディタに近い清潔感のある見た目にします。
- クイズセクションの配色を整理し、正解・不正解の視覚的フィードバックをモダンにします。

## 3. 実装ステップ
1. `src/style.css` のデザイン変数を更新。
2. 各セクションのスタイルを順次修正。
3. ダークモードでの整合性を確認。
